# star_rail_analysis 星穹铁道抽卡分析

基于Vue3和ElementUI开发，通过连接分页读取抽卡数据，并统计出各个卡池4星/五星/已垫抽数/平均五星抽数

后续可能更新：使用图表进行数据分析，导出数据，导入本地数据，使用米哈游token获取authKey......
